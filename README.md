# XMULostAndFind

一个为厦门大学(XMU)打造的智能失物找-寻平台，旨在解决校园内失物招领处**信息分散、寻物效率低下**的核心痛点。

---

## 💡 项目解决了什么？ 

我们通过一个**统一的智能平台**，整合了分散在校园各处的失物招领点的信息，让失物信息发布与寻找变得前所未有的高效。

## ✨ 我们是如何实现的？

我们实现了三大核心功能，将人工智能技术应用于失物检索的全流程：

1.  **多模态语义搜索**
    > 无论是上传**图片**还是输入**文字描述**，系统都能理解其核心语义。

2.  **多维度精准筛选**
    > 在进行智能匹配前，你可以先通过**地点**、**物品分类**、**时间**等传统维度进行快速筛选。这极大地缩小了AI的检索范围，不仅显著提升了寻找效率，也使得匹配结果更为精准。

3.  **关键信息自动标签化**
    > 系统利用OCR技术和图像识别API，自动识别并提取图片中的文字信息（如书名、品牌Logo等）和物品特征、场景作为精确搜索的关键词；同时，CLIP模型会捕捉物品的整体视觉特征，以支持更深层次的语义相似性匹配。
